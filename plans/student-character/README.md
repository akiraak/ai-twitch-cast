## 方針: 目に見えるものから作る

**Step 1〜3 で「2体のアバターが別々の声で喋る」を実現する。**
これが動けば全体設計の成否が早期に確認でき、残りのStep 4〜8は安心して進められる。

```
Step 1: 画面に2体表示   →  目で確認できる
Step 2: 別々に動かせる   →  リップシンク・感情が独立動作
Step 3: 別の声で喋る     →  音で確認できる
─────────────────────────────────
Step 4〜6: バックエンド   →  対話スクリプト自動生成・再生
Step 7〜8: UI・テスト    →  仕上げ
```

## 背景

現在の教師モードは、ちょビが一人で授業を進めるスタイル。先生と生徒の掛け合いがあると:
- **視聴者の代弁者**: 「え、それどういうこと？」を生徒が言ってくれる
- **テンポの改善**: 対話形式で飽きにくくなる
- **理解の促進**: 生徒の質問→先生の補足で重要ポイントを自然に強調

## 生徒キャラの設定

| 項目 | 値 |
|------|-----|
| 名前 | まなび（学び） |
| 性格 | 好奇心旺盛、素直、たまにボケる |
| TTS音声 | Kore（明るい女性声。ちょビのDespinaと差別化）|
| TTSスタイル | 元気で明るい生徒のトーンで読み上げてください |
| VRMモデル | 別途用意（resources/vrm/ に配置）|
| 配置 | 画面左側（先生は右側） |
