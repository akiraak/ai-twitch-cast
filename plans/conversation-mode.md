# 会話モード機能 — 教師モード（第1弾）

## ステータス: 完了

## 背景

通常の配信では、ちょビはチャットコメントに応答する「フリートーク」スタイル。
会話モードを導入することで、特定の目的を持った配信スタイルに切り替えられるようにする。

第1弾として「教師モード」を実装する。教科書の画像やURLをもとに、その内容に沿って授業を進めるモード。

## 確定仕様

- **授業進行**: 自動進行（スクリプト生成→順番に説明）
- **配信画面表示**: アップロード画像は表示しない。授業の進行に合ったテキスト（自由形式）をパネルに表示
- **教え方**: 完全バイリンガル（日英自由に混ぜる、既存言語mix機能活用）
- **チャット対応**: 質問に回答して授業に戻る（脱線最小限）
- **ボリューム**: AIが内容に応じてセクション数を自動調整
- **入力形式**: 画像（複数可）またはURL（テキスト+画像を自動取得）
- **問いかけ**: あり（視聴者に質問→数秒待ち→回答解説）
- **管理画面**: 授業で使用する生成物を全て参照可能にする
- **画像追加**: コンテンツ作成時にも後からでも追加・削除可能
- **スクリプト再生成**: 何度でも可能（上書き）
- **セクション編集**: Phase 1からインライン編集・▲▼並び替え・削除が可能
- **感情（emotion）**: AIが自動で付ける。管理画面では表示のみ（編集不要）

## 管理画面（WebUI）構成

### タブ階層

既存タブに「会話モード」タブを追加。
会話モードタブ内にサブタブでモード種別を切り替える。

```
[配信画面] [キャラクター] [サウンド] [チャット] [TODO] [Debug] [DB] [会話モード]
                                                                        |
                                                              サブタブ: [教師モード] [雑談モード]
```

### 教師モード画面（上下分割レイアウト）

```
┌────────────────────────────────────────────────┐
│ [教師モード] [雑談モード]                        │
├────────────────────────────────────────────────┤
│ [+新規] English 1-1 | 離宮の運動 | TOEIC Part5  │  ← コンテンツ一覧（横並び）
├────────────────────────────────────────────────┤
│                                                │
│ コンテンツ名: [English 1-1              ] [保存] │
│                                                │
│ ▼ 教材ソース (3枚)                              │  ← 折りたたみ
│   [サムネ][サムネ][サムネ]  [画像追加] [URL追加]   │
│                                                │
│ ▶ 抽出テキスト                                  │  ← 折りたたみ（閉じた状態）
│                                                │
│ 授業スクリプト              [スクリプト生成]      │
│ ┌──────────────────────────────────────┐       │
│ │ 1. 🎬 導入  [excited]         [▲][▼][×] │   │  ← 各セクション

│ │ 画面: [Present Tense の作り方... ]     │       │
│ ├──────────────────────────────────────┤       │
│ │ 2. 📖 説明  [neutral]         [▲][▼][×] │   │
│ │ 発話: [三人称単数のとき...       ]     │       │
│ │ TTS:  [...                     ]     │       │
│ │ 画面: [• He goes to school     ]     │       │
│ ├──────────────────────────────────────┤       │
│ │ 3. ❓ 問いかけ [thinking]     [▲][▼][×] │   │
│ │ 発話: [では問題です...          ]     │       │

│ │ A:    [正解は goes です         ]     │       │

│                                                │

└────────────────────────────────────────────────┘
```

#### コンテンツ一覧（上部）
- コンテンツ = 1つの授業単位。名前を付けて複数持てる
  - 例: 「English 1-1」「離宮の運動について」「TOEIC Part5 文法」
- [+新規] ボタンでコンテンツ名を入力して作成
- 横並びタブ or ボタンで選択 → 下部に詳細表示

#### コンテンツ詳細（下部）
- **コンテンツ名**: テキスト入力で編集可能
- **教材ソース**（折りたたみ）
  - アップロード済み画像のサムネイル一覧 + 個別削除
  - [画像追加] ボタン: ファイル選択で画像追加（作成後いつでも）
  - [URL追加] ボタン: URL入力 → Webページからテキスト+画像を自動取得
- **抽出テキスト**（折りたたみ、デフォルト閉じ）
  - 画像/URLから読み取った原文を表示
- **授業スクリプト**
  - [スクリプト生成] ボタン（何度でも再生成可能、上書き）
  - セクション一覧（インライン編集）:
    - 種別ラベル + 感情（表示のみ）
    - 発話テキスト（content）— インライン編集
    - TTS用テキスト（tts_text）— インライン編集
    - 画面表示テキスト（display_text）— インライン編集、自由形式
    - questionセクション: 問いかけ/回答/待ち秒数 — インライン編集
    - [▲][▼] 並び替え、[×] 削除
- **操作ボタン**: [授業開始] / [コンテンツ削除]
